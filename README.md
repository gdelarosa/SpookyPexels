### SpookyPexels
![giphy](https://media.giphy.com/media/nkBa9eREC4D7O/giphy.gif)

A simple Halloween themed iOS app that displays spooky images. 

# Uses: 
- 🎃 Swift 5
- 💀 Pexels API
- 👻 MVVM design pattern 
- 🧟‍♀️ UIKit 
- 🩸 Programmatic layout
- 🧛🏼‍♀️ Compositional layout
- 👹 Codable 
- 😈 URLSession 
- 👽 Unit tests 
- 👾 Animations 
- 👺 Grand central dispatch
- ⚰️ Error handling
