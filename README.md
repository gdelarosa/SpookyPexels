### SpookyPexels
![giphy](https://media.giphy.com/media/nkBa9eREC4D7O/giphy.gif)

A simple Halloween themed iOS app that displays spooky images. 

# Uses: 
- 🎃 Swift 5
- 💀 Pexels API
- 👻 MVC design pattern 
- 🧟‍♀️ UIKit 
- 🧛🏼‍♀️ CollectionView with compositional layout
- 👹 Codable 
- 😈 URLSession 
- 👽 Unit Tests 
- 👾 Animations 
- 👺 Grand Central Dispatch
- ⚰️ Error Handling
